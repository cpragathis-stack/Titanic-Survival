# Titanic Survival Prediction using Machine Learning

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning classification algorithms.

## Dataset
Titanic Dataset from Kaggle:
https://www.kaggle.com/datasets/yasserh/titanic-dataset

## Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Steps
1. Data Loading and Exploration
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Best Model Analysis

## Models Used
- Logistic Regression
- Random Forest Classifier
- K-Nearest Neighbors (KNN)

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix

## Results
Random Forest achieved the highest accuracy and was selected as the best-performing model.
